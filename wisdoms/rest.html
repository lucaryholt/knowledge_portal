REST: Representational State Transfer


General

RESTFul web services are services via HTTP requests.

The services make use of the HTTP request methods GET, PUT, DELETE & POST.
This enables us to have multiple services on the same end point/path. You can have the service return a list, when a user sends a GET request, replace a resource when a user sends a PUT request and so on.


Insomnia & Postman

Insomnia and Postman are applications designed for testing RESTFul web services.
They enables the user to send HTTP requests and see the response. This allows for quick debugging of REST services without writing the software requesting and receiving.
