Collection that stores multiple values.

General syntax

Instantiate array:

const array = ['Item one', 'Item two', 'Item three']

You can also instantiate an array using the JS 'new' keyword:

const array = new Array('Item one', 'Item two', 'Item three')

Getting item from array:
array[1] returns 'Item two' as arrays are zero indexed.
You can access the last element in an array with: array[-1].

Updating item in array:
We can change 'Item three' to 'Item 3' with:

array[2] = 'Item 3'

Adding item to array:

array.push('Item four')

Removing last item in array:

array.pop()

Types

As said earlier, arrays can contain different types, so you could have an array like this:

const misc = ['this is a string', 80, function(){console.log('Print statement')}]

You could then call misc[2]() and 'Print statement' would be printed to the console.

This is not good coding standard, but it is possible.
Normally you would declare the function elsewhere and then reference it in the array.

Properties and methods

Arrays have built in properties and methods:

Get the length of an array (useful in for loops):

array.length

Sort an array:

array.sort()
(See also 'Sorting JSON by specific value').
The sort method sorts the original array.

Get index of specific element in array:

array.indexOf('Item two')
would return 1.

Splice method:
You can add items to an array at an index you choose with array.splice(start, deleteCount, items).
The 'start' parameter indicates where in the array you want to start the operation.
'deleteCount' is how many items you want to delete (0 if you just want to insert). Can be used to overwrite items in array.
'items' is the items you want to insert.
Example:

array.splice(1, 0, 'Inserted item', 'Another')
would return an empty array, but 'array' would now look like this: ['Item one', 'Inserted item', 'Another', 'Item two', 'Item three'].
This can also be used to delete items.
This method returns the deleted items.

Looping through array elements

Syntax for a for-loop that prints out all elements in array:

for(let i = 0; i < array.length; i++){
    console.log(array[i]);
}

Copying array

If you want to copy an array, you can not just do copyArray = array, as this will copy the reference and both 'copyArray' and 'array' will point to the same array.
Here you can use the slice() method, as this copies the value of the array instead of the reference:

copyArray = array.slice()
