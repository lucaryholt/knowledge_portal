Stores multiple values. Can be different types.

Can be used if you want to loop through a collection of variables.


General syntax

Syntax: const array = ['Item one', 'Item two', 'Item three'].

You can also instantiate an array using the JS 'new' keyword:
const array = new Array('Item one', 'Item two', 'Item three');

Getting item from array:
array[1] returns 'Item two' as arrays are zero indexed.

Updating item in array:
We can change 'Item three' to 'Item 3' with: array[2] = 'Item 3'.


Types

As said earlier, arrays can contain different types, so you could have an array like this:

const misc = ['this is a string', 80, function(){console.log('Print statement')}].

You could then call misc[2]() and 'Print statement' would be printed to the console.

This is not good coding standard, but it is possible. Normally you would declare the function elsewhere and then reference it in the array.
