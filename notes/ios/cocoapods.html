CocoaPods is a dependency manager for Swift and Objective-C projects.

Usage

If you want to work with, for instance, Google Firebase in an iOS app, you could use CocoaPods to install those dependencies.

Podfile

To define what pods to install, you need to create a file simply called Podfile. This file contains what pods you want to install with the pod install command. 

    # Uncomment the next line to define a global platform for your project
    # platform :ios, '9.0'

    target 'MyNotes' do
      # Comment the next line if you don't want to use dynamic frameworks
      use_frameworks!

      # Pods for MyNotebook
      pod 'Firebase/Analytics'
      pod 'Firebase/Firestore'
      pod 'Firebase/Storage'
    end
