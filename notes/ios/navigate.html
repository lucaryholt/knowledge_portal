Show the user different views

Navigation

A key part of almost every app, is the ability to show different information and screens, depending of the needs of the user.
A way to do this in SwiftUI is by using NavigationView and NavigationLink components. 

    import SwiftUI

    struct ContentView: View {
        var body: some View {
            NavigationView{
                VStack{
                    NavigationLink(
                        destination: DetailView(image: Image("image0")),
                        label: {
                            Text("Show image 0")
                        })
                    NavigationLink(
                        destination: DetailView(image: Image("image1")),
                        label: {
                            Text("Show image 1")
                        })
                    NavigationLink(
                        destination: DetailView(image: Image("image2")),
                        label: {
                            Text("Show image 2")
                        })
                }
            }
        }
    }
This would show three buttons (in a VStack) that would go to DetailView, displaying each of these images.
