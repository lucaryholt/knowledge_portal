JavaScript FileSystem

General

fs is commonly used to:
  • Read files
  • Create files
  • Update files
  • Delete files
  • Rename files
It is imported via the require() function.
fs is built into Node.JS, so there is no need to install it separately (via npm).

Read Files

The fs.readFile() function is used to read files from a computer.
You can also use the fs.readFileSync() to synchronously read the file.
The fs.readFile() reads the file asynchronously.

Create Files

fs has three functions for creating new files:
  • fs.appendFile()
  • fs.open()
  • fs.writeFile()
fs.appendFile() appends content to a file. If that file does not exist, it will be created.
fs.open() takes a flag as second argument, if that flag is set to 'w', the file will be opened for writing. If the file does not exist, an empty file will be created.
fs.writeFile() replaces the specified file and contents, if it exists. If it does not a new file with the specified content will be created.

Update Files

fs has two functions for updating files:
  • fs.appendFile()
  • fs.writeFile()
fs.appendFile() appends the specified content at the end of the file.
fs.writeFile() replaces the specified file and contents.

Delete Files

To delete files with fs we can use fs.unlink().

Rename Files

To rename files with fs we can use fs.rename().
