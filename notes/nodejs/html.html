Hypertext Markup Language

General HTML

The standard language for documents designed to be displayed in a web browser.
HTML is usually styled with CSS (Cascading Style Sheets) and scripted with JS.
HTML files have the file extension .html (or .htm).
The latest version of the language is HTML5.
In HTML you can place various elements. The elements generally have an start <script> and end </script> tag. The elements can be placed within each other and thereby create a hierarchy of elements.
A normal HTML5 page has the following syntax: 

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>Title</title>
    </head>
    <body>

    </body>
    </html>
The document starts with the <!DOCTYPE html> declaration. All HTML documents must have this. This is a message to the browser about what kind of document to expect.
Under this declaration we see a number of different elements. The <html lang="en"> encloses the content of the document. Inside that we have the <head> and <body> elements.
The <head> element contains some information about the document, including title. This title is what we see in the browser tab. You can also place <link> and <script> elements here to import, as an example, CSS and JS files. If placed in the <head> element the browser will load these files, before showing the content.
The <body> element is the content that is actually shown in the browser. This could be text, images and a lot more.
As you can in the <head>, you can also import CSS and JS here. If these are placed here, the browser can show the HTML before actually downloading the files.
When the browser downloads a JS file with the <script> element, it loads the script and runs it like a normal JavaScript file. If there are declared functions in the script, these can now be called by the onClick property (among others) on an HTML element.
Usually when using JS in a browser, it is to dynamically updating the HTML. The most used library for this is jQuery.

DOM

DOM is short for Document Object Model.
We use this when updating HTML with jQuery.
The HTML DOM is constructed as a tree of Objects (image below).
When the browser loads a webpage it creates a Document Object Model.
When we use jQuery we use this to select which element(s) to manipulate (or read information from).
The HTML DOM defines:
  • HTML elements as objects
  • Properties of all HTML elements
  • Methods to access all HTML elements
  • Events for all HTML elements
"The HTML DOM is a standard for how to get, change, add, or delete HTML elements."
(quote from 'W3 Schools - JavaScript HTML DOM').

CSS Styling

There are three different approaches to styling HTML with CSS:
  • Inline
  • Internal
  • External
Inline styling is when the styling is written as an attribute of the HTML element: 

    <h1 style="background: pink">Title</h1>
With internal styling, the CSS is written in a <style> tag in the <head> element.
External styling is one of the more commonly used approaches. Here all the CSS styling is moved to separate stylesheet. The stylesheet is then imported like an external JS script, but in a <link> tag instead: 

    <link rel="stylesheet" href="style/style.css">

Bootstrap

Bootstrap is a HTML, CSS and JS library that provide easy styling and building of a website. A way to use Bootstrap is to refer to the Bootstrap library hosted on a CDN just like jQuery.

Events

There exists a number of event attributes (also called event listeners) you can apply to HTML elements. These events have the ability to trigger actions in the browser. This action is often JavaScript code.
Some event attributes worth noting are onclick, onblur and onfocus.
onclick activates when a user clicks on the element. onblur and onfocus are form event attributes. onblur activates when an element loses focus and onfocus, does exactly what it sounds like.
There are window, form, keyboard, mouse, drag, clipboard, media and misc events. A full list in the W3Schools - HTML Event Attributes link.
