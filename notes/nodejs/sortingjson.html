This is just a way of sorting a List of JS/JSON objects that I came across and it seemed useful

Usage

Let's say we have an array of JS/JSON object like this:
const array = [{ title : 'Tenet', score : 7.8 }, { title : 'Interstellar', score : 8.6 }]; and we want to sort it by the title property. If we just call array.sort(), we get the exact same array. We have to modify the sort method a bit:
array.sort(function (a, b){
&*return a.title.localeCompare(b.title);
}); Here we pass a function, that we define, as a parameter to sort. This function uses the string method localeCompare(). Now our array looks like this:
[ { title: 'Interstellar', score: 8.6 }, { title: 'Tenet', score: 7.8 } ];
