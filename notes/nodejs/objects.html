In JS variables are containers for data. JS Objects are also variables, but can contain many values

General

Syntax for defining a JS object:
const objectDemo = { type : 'Demo', isThisEnoughDemoData : true, demoFunc : function () { console.log('404') } }; Spaces and line breaks are not important, the above example, could also be written like this:
const objectDemo = {
&*type : 'Demo',
&*isThisEnoughDemoData : true
};
The KeyValuePairs in JS objects are called properties. You can save every data type as a property in an object.

Accessing Object Properties

There are two methods for accessing a JS Object property.
The dot-notation:
objectDemo.type; and the square brackets-notation:
objectDemo["isThisEnoughDemoData"]; The norm is to use dot-notation.
If you need to call a method in an object, you can do the following: objectDemo.demoFunc();
