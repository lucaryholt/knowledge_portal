Swift variables can hold many data types

Types

Basic data types:
  • Int & UInt
  • Float
  • Double
  • Bool
  • String
  • Character
  • Optional
  • Tuples
Most of these are self-explanatory and function much like most other programming languages. A few of them will be explained more below.

Optional

This represents a variable that either holds a value or no value.
When declaring a variable, you can make it optional by using the ? symbol: 

    var perhapsInt:Int?
This essentially sets the variable to nil.
This is used a lot when a class holds a variable, that is not used all the time. Could be a document, where an image is attached optionally. Here you would enable the user to instantiate a document without an image.
When you need to use an optional variable, you need to unwrap it, as the optional data type basically holds the value. There are different approaches to unwrapping.
The most straight-forward unwrapping is the force unwrapping method: 

    perhapsInt!
When using this method, it is imperative that the optional has an value or it will cause an exception being thrown or the app crashing.
This method is not used unless there is an explicit reason to do so.
Normally when unwrapping an optional value, you will test if there actually is an value before using it. This can be done various ways.
One way to do it: 

    if myString != nil {
        print(myString)
    } else {
        print("myString has nil value")
    }
Another: 

    if let int = perhapsInt {
        int
    }
The first method checks if the optional is not nil and then does something with it. The second checks if you can save the optional in another variable, meaning if it has an value, if it can then you can use that new variable.
They do basically the same, just use whatever suits you.
