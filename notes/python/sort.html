Sort data

sorted()

Sorting numbers
A simple way to sort an list of numbers: 

    numbers = [6, 9, 3, 1]
    sorted(numbers)
This would result in a new, sorted list. The original list is unchanged.
This can also be used on tuples and sets.
Sorting strings
If you use sorted() on a string, it treats it like a list of characters and returns a new list of the characters sorted. 

    string_value = 'I like to sort'
    sorted_string = sorted(string_value)
Non-Comparable Data Types
If you try to sort a list containing Non-Comparable types, it throws a TypeError.
reverse argument
You can use the reverse argument, to get the values sorted in reverse: 

    numbers = [6, 9, 3, 1]
    sorted(numbers, reverse=True)
key argument
This argument expects a function being passed. That function will then be used on every value in the list, to determine how to sort it.
A basic example of this is to use len: 

    words = ['banana', 'pie', 'Washington', 'book']
    sorted(words, key=len)
This now sorts the list, not alphabetically, but according to the length of each string.
This can be any function, that returns a value.

sort()

sort() is a method of the list class and can only be used with lists.
sort() does not return anything, but modifies the values in place.
You don't pass a list to sort(), but simply call it on a list.
