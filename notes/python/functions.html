A function is a block of code which only runs when it is called.
You can pass data, known as parameters, into a function.
A function can return data as a result

Syntax

To define a function in python you use the def keyword: def hello_world():
&*print('Hello!!') To call it you would then write: hello_world() which would print Hello!!.

Arguments

You can pass information to a function with arguments: def hello(name):
&*print('Hello ' + name + '!!') and then pass information to it: hello('Bob') which would print Hello Bob!!.
You can only pass as many arguments as is specified in the function definition, but you can specify an unknown number of arguments in the definition.
To do this you add a * before the parameter name: def hello(*names):
&*print('Hello ' + str(names) + '!!') This would not produce a very pretty result as converting a tuple to string does not come out clean. But now you can write an undefined number of parameters to the hello() function.
When passing arguments to a function you can also name the arguments. So if we had a function as this: def hello(name1, name2):
&*print('Hello ' + name1 + ' and ' + name2) You could call it like this: hello(name2 = 'Torp', name1 = 'Sko') and it would print: Hello Sko and Torp.
Just as you can define an unknown number of unnamed arguments, you can also define an unknows number of named arguments. To do this we use **: def hello(**names):
&*print('Hello ' + names["best_name"] + '!!') This will pass a dictionary to the function, which then can be used to extract the named arguments: hello(worst_name = 'Torp', best_name = 'Sko') Here we pass two arguments called worst_name and best_name. In the function body we only use the best_name argument, but as we defined **names, then we are allowed to do this and as a result Hello Sko!! is printed.
You can also define a function with an argument, but have that argument have an default value: def hello(name = "Arnold"):
&*print('Hello ' + name + '!!') This means that if someone where to call hello() without a name argument Arnold will be used instead.
In the above examples we have only passed strings to the functions, but you can pass almost anything as an argument.

Return value

To define a return value you simply use the return keyword: def hello(name):
&*return 'Hello ' + name + '!!' This would now instead of printing the string, return it.

Pass

If you for some reason have a function without a body, python will not allow your script to compile. To combat this, you can use the pass keyword: def hello(name):
&*pass

Recursion

Python allows for recursive functions. To define a recursive function, you simply call it from itself, just as you would have called it normally.
