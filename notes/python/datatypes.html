Python has a number of built-in data types

Categories

There are a number of different categories of data types:
  • Text type:
    • str
  • Numeric types:
    • int
    • float
    • complex
  • Sequence types:
    • list
    • tuple
    • range
  • Mapping type:
    • dict
  • Set types:
    • set
    • frozenset
  • Boolean type:
    • bool
  • Binary types:
    • bytes
    • bytearray
    • memoryview

Getting the data type

You can use the built-in function type() to get the data type. 

    x = 5
    print(type(x))
This would print <class 'int'>.

String

Strings in python are surrounded by either single or double quotation marks "hello" and 'hello' are the same thing.
You can make multiline strings by using three quotation marks: 

    a = """Lorem ipsum dolor sit amet,
    consectetur adipiscing elit,
    sed do eiusmod tempor incididunt
    ut labore et dolore magna aliqua."""
In Python strings are arrays of bytes, representing unicode characters (Python does not have a character type, a single character is simply a string with the length of 1). This means that we can treat a string like an array and access characters in the string with brackets: 

    a = "Hello, World!"
    print(a[1])
This would print e. We can also loop through a string exactly like we would an array.
Built in methods
Strings have a number of built in methods
  • upper() returns the string in upper case
  • lower() returns the string in lower case
  • strip() removes whitespace before and after the text
  • replace() takes two strings as argument and replaces the first string with the second
  • split() takes a string, and returns a list of substrings, if it finds the separator

Integer

Int, or integer, is a whole number, positive or negative, without decimals, of unlimited length.

Float

Float, or "floating point number" is a number, positive or negative, containing one or more decimals.
Float can also be scientific numbers with an "e" to indicate the power of 10.

Complex

Complex numbers are written with a "j" as the imaginary part.

Boolean

Booleans represent one of two values: True or False.

range

Built-in function or data type?
In Python 2, if you wrote type(range) you would get the result <type 'builtin_function_or_method'>, where in Python 3 you get <class 'type'>, which suggests that range has been classified as a type in Python 3.
Furthermore, if you check the type when you actually use range, like this in Python 2 type(range(10)) you get <type 'list'> and in Python 3 <class 'range'>.

frozenset

frozenset() is a function that returns an unchangeable frozenset object, which behaves like a set object, only unchangeable. It takes an iterable as parameter.

Bytes

The bytes() method returns a immutable bytes object initialized with the given size and data.
It takes three optional parameters:
  • source - source to initialize the array of bytes
  • encoding - if the source is a string, the encoding of the string
  • errors - if the source is a string, the action to take when the encoding conversion fails

Bytearray

The bytearray() method returns a bytearray object which is an array of the given bytes. This is mutable.
It takes three optional parameters:
  • source - source to initialize the array of bytes
  • encoding - if the source is a string, the encoding of the string
  • errors - if the source is a string, the action to take when the encoding conversion fails

Memoryview

The memoryview() function returns a memory view object of the given argument.
Python Buffer Protocol
The buffer protocol provides a way to access the internal data of an object. This internal data is a memory array or a buffer.
The buffer protocol allows one object to expose its internal data (buffers) and the other to access those buffers without intermediate copying.
This protocol is not accessible in our normal codebase. In order to expose this protocol to the normal Python codebase, memory views are present.
What is a memoryview?
A memory view is a safe way to expose the buffer protocol in Python. It allows you to access the internal buffers of an object by creating a memory view object.
