The requests library is the de facto standard for making HTTP requests in Python

General

The requests package is installed with pip and imported like any other package.

GET Request

To get data via a HTTP request, we use the GET method. 

    response = requests.get('https://api.github.com')
This method returns the response from the GET request to the specified URL.
You can then access the content of the response, simply by accessing .content on the response object. This gives you the content as bytes, but you can get it as a string, by accessing .text instead. You can also use .json(), that returns a dictionary.
Query Strings
You can also specify query strings: 

    response = requests.get(
        'https://api.github.com/search/repositories',
        params={'q': 'requests+language:python'},
    )
Headers
And headers: 

    response = requests.get(
        'https://api.github.com/search/repositories',
        params={'q': 'requests+language:python'},
        headers={'Accept': 'application/vnd.github.v3.text-match+json'},
    )

POST, PUT, PATCH and others

To use methods, where you need to pass data (like POST), you use the data argument: 

    requests.post('https://httpbin.org/post', data={'key':'value'})
