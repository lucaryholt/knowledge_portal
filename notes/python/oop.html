Object-Oriented Programming in Python

Define a class

To define a class in python we use the class keyword and this needs a initialisation method: 

    class Dog:
        def __init__(self, name, age):
            self.name = name
            self.age = age
This takes two arguments and sets them as instance attributes for this class. The self argument is needed in any method that is called outside of the class.

Instantiate an object

Instantiate a object from a class: 

    buddy = Dog('Alfred', 7)
Now we can access the values with dot notation, for instance buddy.name returns Alfred.

Inherit from other classes

You can make child classes from parent classes: 

    class Dog:
        def __init__(self, name, age):
            self.name = name
            self.age = age

    class Bulldog(Dog):
        pass

    bully = Bulldog('Jim', 1)
This makes the Bulldog class inherit the properties from the Dog class. You can also add or overwrite properties and behavior in child classes.
