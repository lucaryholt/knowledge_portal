Loop through data

For loops

For loops in Python function like for each loops in something like Java - collection-based iteration. A Python for loop looks like this: 

    for <var> in <iterable>:
        <statement(s)>
A use example: 

    a = ['foo', 'bar', 'baz']
    for i in a:
        print(i)
This would print the three values in a.
To emulate a for i loop, you can use the range() built in function. 

    for i in range(10):
        print(i)
break and continue
You can use the keywords break and continue in loops. break terminates the loop completely, while continue terminates the current iteration and proceeds to the next.
else
The else clause can also be used with loops. This clause will be executed if the iterable is exhausted, but not if the loop is broken by a break statement. 

    for i in ['foo', 'bar', 'baz', 'qux']:
        if i == 'bar':
            break
        print(i)
    else:
        print('Done') # Will not execute

While loops

A while loop in Python: 

    while <expr>:
        <statement(s)>
A use example: 

    n = 5
    while n > 0:
        n -= 1
        print(n)
break and continue
The two keywords have the same functionalities as in the for loop, but while continue simply runs the next iteration in a for loop, here it evaluates the statement again (as does every iteration in a while loop).
else
The else clause functions the exact same way as with for loops.
Infinite loops
You could write a infinite while loop like this: 

    while True:
        print('Still true')
Could be used to run a service that, for instance, continually checks for some condition - maybe with a sleep function to run it at intervals.
An infinite loop can still be broken with a break keyword.
