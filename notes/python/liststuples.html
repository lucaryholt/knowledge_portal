Lists

General
A list is a collection of objects, like a list in other programming languages, but more flexible.
Syntax
You define a list with []:
a = ['a value', 'another value', 'wait', 'also a value']
Characteristics
  • Ordered
  • Can contain any object. Lists can consist of a collection of the same type of object, or a mix of all kinds. They can even contain functions, classes, modules and more.
    Can also contain any number of objects, from 0 to as many as the computers memory will allow.
  • Can be accessed by index
  • Can be nested
  • Mutable
  • Dynamic
To access an element in a list, you use []:
a[0] = 'now it is this value'
You can also easily access the last value in a list:
a[-1] = 'last value'


Slicing
For this section we will work with a list that looks like this:
a = ['foo', 'bar', 'baz', 'qux', 'quux', 'corge']
You can use slicing to manipulate lists and extract the parts of the list that you need.
The expression a[m:n] will return a portion of the list, from index m to, but not including, n. This means that
a[2:5]
Returns ['baz', 'quz', 'quux'].
When slicing you both use positive and negative indices.
More on slicing can be found in the 'Real Python - Lists and Tuples in Python' link below.
Operators and built-in functions
You can use a number of Python operators and built-in function on lists. Like the in and not in operators:
'qux' in a
Would return True.
You can also use the operators + and *, as well as the len(), min() and max() functions.

Tuples

General
Tuples are identical to lists, except for:
  • Tuples are defined with (), instead of [].
  • Tuples are immutable.
Why use a tuple?
If the only differences were the two above, you might not see the reason to use tuples over lists. Here are some reasons:
  • Execution of code manipulating tuples are faster than with a equal list.
  • Protection of data. Sometime you want data to be immutable.
  • In the Python data type dictionary, requires one of it's components to be immutable. Here you can use a tuple.

List Comprehensions

List comprehensions provide a concise way to create lists.
The way to make a list comprehension, it to use brackets, with an expression followed by a for clause, then optional for or if clauses.
An example of this can be the following code for creating a list:
new_list = []
for i in old_list:
    if filter(i):
        new_list.append(expressions(i))
This can be written with a list comprehension:
new_list = [expression(i) for i in old_list if filter(i)]
Another example:
l = []
for i in range(1, 10):
    l.append(i)
Could be written:
[i for i in range(1, 10)]
