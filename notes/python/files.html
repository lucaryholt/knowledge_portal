Python has several built-in modules and functions for handling files

Open files

Reading and writing data to files in python are quite easy to do. You use the open() method. This method takes to arguments, a file name and a flag specifying which mode to open the file in. with open('data.txt', 'r') as f:
&*data = f.read() Here we open the file data.txt and save the contents in the variable data.
We passed the r flag to open the file in read only mode. with open('data.txt', 'w') as f:
&*f.write(data) Here we instead passed the w flag, which opens the file in write mode.
In the two code snippets above we have used pythons with open(...) as pattern.

Directories

You can use the os module to scan directories and get the result returned: import os

with os.scandir('my_directory/') as entries:
&*for entry in entries:
&*&*print(entry.name) There are different methods and modules that can scan directories, some are os.scandir(), os.listdir() and pathlib.Path.iterdir(). They each do slightly different things.
You can create a directory with the os.mkdir() function. It takes a directory name as argument. If the directory already exists, os.mkdir() raises FileExistsError.
To make multiple directories you can use os.makedirs(), which can be used to make directory trees. os.makedirs('trunk/branch/leave') This function call would make a leave directory, inside a trunk directory, inside a trunk directory.
This function also takes an optional mode argument, which specifies the permissions of the directories.

Deleting files and directories

There are a number of different ways to delete files and directories in python.
Two, used for deleting files, are os.remove() and os.unlink(). Both functions take an path to a file and thereafter delete them.
To delete a directory, you can use os.rmdir(). This function takes a path to a directory and deletes it.
This will throw an error if the directory is not empty.
To delete an non-empty directory you can use the rmtree() function from the shutil module.
