Dictionaries and Sets are collections in Python

Dictionary

Unordered, changeable and indexed.
To define a dictionary we use the following syntax: 

    my_dict =	{
        "brand": "Ford",
        "model": "Mustang",
        "year": 1964
    }
Accessing element in a dictionary: 

    x = my_dict["brand"]
To change value you do: 

    my_dict["brand"] = "Mazda"
You can easily loop through a dictionary, like you would a list: 

    for x in my_dict:
        print(x)
You can use the in keyword, to check for keys in a dictionary: 

    if "brand" in my_dict:
        print("it contains brand!")
The built-in Python function len() also works on dictionaries.
To add an item, you use the same syntax as when changing a value: 

    my_dict["hp"] = 200
To delete an item you use the pop() method: 

    my_dict.pop("hp")
You can also use popitem(), which removes the last inserted item.
The del keyword can also be used: 

    del my_dict["hp"]
To copy a dictionary you can use the copy() method. This will copy the value, and not the reference.
Nested dictionaries can be done easily: 

    cars = {
        "car_0" : {
            "brand": "Ford",
            "model": "Mustang",
            "year": 1964
        },
        "car_1" : {
            "brand": "VW",
            "model": "Golf",
            "year": 2019
        }
    }

Set

Unordered and unindexed.
Define a set: 

        my_set = {"pengu", "donald", "gurli"}
You can not access items in a set, by referencing a key or index, but you can loop through it or ask if a value is present: 

    for x in my_set:
        print(x)

    if "pengu" in my_set:
        print("pengu is here!")
You can not change items in a set, but you can add them using the add() or update() (to add more than one item) methods: 

    my_set.add("goofy")

    my_set.update(["tintin", "haddock", "terry"])
The built-in Python function len() also works on sets.
To remove items in a set, you use the discard() method: 

    my_set.discard("goofy")
If the specified item does not exist, discard() will not raise an error.
You can also use the pop() method. This removes the last item, but as sets are unordered, you can not know which item will be removed. pop() returns the removed item.
To clear the set itself, you can use clear() and to delete it, use the del keyword.
You can join two sets with the union() method: 

    set1 = {"a", "b" , "c"}
    set2 = {1, 2, 3}

    set3 = set1.union(set2)
The intersection() method can be used to find the elements that exist in two or more sets: 

    x = {"apple", "banana", "cherry"}
    y = {"google", "microsoft", "apple"}

    z = x.intersection(y)
Here z will reference a set containing the element apple.
On the other hand, the difference() method finds the elements that exist in one set and not another: 

    x = {"apple", "banana", "cherry"}
    y = {"google", "microsoft", "apple"}

    z = x.difference(y)
Here z will reference a set containing the elements banana and cherry.
