Lambda expressions can help you remove boilerplate code from your projects and process huge amounts of data with ease.

General

Lambda expressions are like closures in Swift and callbacks in JavaScript.
They allow us to define functions in a simple syntax and pass them as parameters. This is often used to perform asynchronous actions.

Syntax

The syntax for defining a lambda expression is: 

    (parameter1, parameter2) -> { code to be executed }
If you want to pass a function before passing it, you can use a short-hand: 

    TodoAdapter todoAdapter = new TodoAdapter(this, toDoRepo.getItems(), this::onResume);
In this example the last parameter is a reference to a function defined in the class that we instantiate the TodoAdapter. This could also have been written like this: 

    TodoAdapter todoAdapter = new TodoAdapter(this, toDoRepo.getItems(), () -> {
        onResume();
    });
