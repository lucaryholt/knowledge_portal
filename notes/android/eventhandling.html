Events are a useful way to collect data about a user's interaction with interactive components of Applications

onClick

On all GUI elements you can set a property called onClick. This can be filled out with a Java function, following a certain pattern. It needs to be a public void method, with a parameter View view: 

    public void click(View view){
        System.out.println("Clicked!");
    }
Android Studio automatically detects these and presents them in the onClick property.

ListView Item Click Listener

When using a ListView element, you can set a click action. To do this we use the setOnItemClickListener method on the ListView. This method accepts a lambda expression with four parameters:
  • _listView
  • linearLayout
  • adapterPos
  • arrPos
The arrPos is the index of the item the user clicks/presses in the GUI. We can use this to access the element and process it. A simple console print example: 

    listView.setOnItemClickListener((_listView, linearLayout, adapterPos, arrPos) -> {
        System.out.println("Clicked element on row: " + arrPos);
    });
