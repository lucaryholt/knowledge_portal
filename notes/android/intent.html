General

To navigate between Activities in Android we use Intent.

Syntax

A basic intent that just redirects to other Activity: 

    Intent intent = new Intent(this, Activity2.class);
    startActivity(intent);
To go back to the previous Activity we simply use the finish() method: 

    public void home(View view){
        finish();
    }

Passing data with Intent

To pass data with intent, we inject data as KeyValuePairs: 

    Intent intent = new Intent(this, Activity2.class);

    intent.putExtra("message", "Hey you!");

    startActivity(intent);
This code snippet, would store the value Hey you! with the key message.
To extract it in the target activity: 

    String content = getIntent().getStringExtra("message");
Here we use the key to extract the value. Now the String content, will contain Hey you!. We use the method getIntent() and on that getStringExtra(key). There exists a whole list of other methods to extract other data types.
